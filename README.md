# Gotly AI

A lightweight AI-powered capture tool.

Turn one sentence into structured notes, todos, and bookmarks — instantly.

No categories. No forms. No friction.  
Just say it. It’s saved.

---

## ✨ Why Gotly AI

Most tools require you to think before you write.

- Which type is this?
- Where should it go?
- Should I tag it?

Gotly AI removes that friction.

Just type one sentence.  
Gotly understands your intent and organizes it for you.

---

## ⚡ What you can do

- Capture todos  
  > "Remind me to send the proposal tomorrow"

- Save quick notes  
  > "Idea: build a lightweight AI capture tool"

- Store bookmarks  
  > "https://example.com useful article"

- Ask naturally  
  > "What do I need to do today?"

---

## 🧠 How it works

Gotly AI turns natural language into structured data.

- Intent detection (todo / note / bookmark)
- Time parsing (dates, reminders)
- Lightweight storage (local-first friendly)
- Fast retrieval via simple queries

Instead of forcing structure, Gotly generates it for you.

---

## 🚀 Core principles

- **Frictionless input** — no thinking, just capture  
- **AI-assisted structure** — automatic organization  
- **Lightweight by design** — no complex systems  
- **Daily-use focused** — built for speed, not features
